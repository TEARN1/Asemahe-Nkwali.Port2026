# Asemahle Nkwali - Logistics Technology Portfolio (2026)

A high-performance, immersive 3D portfolio showcasing the intersection of Supply Chain Management (SCM) and modern Frontend Engineering.

## 🚀 Key Features
- **3D Interactive Environment**: Built with Three.js featuring a dynamic starfield and "Warp Speed" wind effects.
- **Logistics DNA**: Integrated KPIs (Warehouse Dwell Time, Inventory Accuracy) and Project categorization (Primary vs Secondary Distribution).
- **Gatekeeper Security**: Integrated lead-capture system using EmailJS to authenticate visitors and track recruiter engagement.
- **AI-Powered Interaction**: Custom chatbot with "Quiz Mode" to test knowledge of logistics and frontend terminology.
- **Performance Optimized**: 60FPS animations and responsive design for all devices.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **3D Engine**: Three.js
- **Lead Capture**: EmailJS
- **Analytics**: Custom Session Tracking API
- **Audio**: Web Audio API

## 📈 Impact Metrics
- **18% reduction** in warehouse dwell time (Project Alpha).
- **40% reduction** in decision latency (Project Beta).
- **99.5% inventory accuracy** (Enterprise WMS integration).
